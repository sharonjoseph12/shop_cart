# ShopSphere Logistics

ShopSphere is a premium frontend-only e-commerce and logistics platform built with React and Vite. It features role-based access for Customers, Sellers, and Delivery Partners, all managed efficiently through Context API and Local Storage.

## Features

- **Customer Portal**: Browse products, search, filter, and place orders.
- **Seller Dashboard**: Manage product inventory and track revenue metrics.
- **Delivery Partner Dashboard**: View assigned deliveries and a React Flow visualization of delivery routes optimized using **Kruskal's Minimum Spanning Tree** algorithm.

## Technology Stack

- React 19
- Vite
- React Router DOM
- Tailwind CSS
- Framer Motion
- Recharts & React Flow

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Login as a Customer, Seller, or Delivery Partner by selecting the respective role on the Login page. Any email can be used for simulation.
